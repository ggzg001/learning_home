## 斐波那契数列

```python
def febonaqi(n):
    """

    :param n: 斐波那契数列的第几个元素
    :return:
    """
    if n == 1 or n == 2:
        return n - 1
    else:
        return febonaqi(n-1) + febonaqi(n-2)
for i in range(1, 10):
    print(febonaqi(i), end=" ")
```

## 二分法查询

```python
def bisection(lis, data, count = -1):
    """
    二分法查询
    :param lis:
    :param data:
    :param count: 查询的次数
    :return:
    """
    count += 1
    if not lis:
        print("data not in lis", count)
        return
    middle = len(lis) // 2
    if lis[middle] == data:
        print("data in list")
    elif lis[middle] > data:
        bisection(lis[:middle], data, count)
    else:
        bisection(lis[middle+1:], data,count)
```
